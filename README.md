# 3D-Printing-Tools
Tools for 3D printing navisworks items in blender

import_cleanup.py instructions:
1. Open Blender and select Edit -> Preferences and the select Add-ons
2. Select the down arrow in the top right of the screen and select "Install from Disc..." and select import_cleanup.py
3. Close the Preferences tab and click N in the Blender Scene to open the Add-ons menu
4. Select Navisworks and choose a collection name and center point
5. Select "Import and Clean FBX" and then select the FBX file you want to import
